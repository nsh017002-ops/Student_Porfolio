# Niharika Sharma — Professional Portfolio 2026

A high-performance, single-page professional portfolio built with a focus on "Glassmorphism" aesthetics and fluid user experience. This site showcases the technical expertise, projects, and academic background of Niharika Sharma.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom layouts using CSS Grid and Flexbox.
- **JavaScript (Vanilla)**: For custom interactions and scroll animations.
- **Intersection Observer API**: To trigger "reveal" animations as the user scrolls.
- **Google Fonts**: Playfair Display (Serif), DM Sans (Sans), and JetBrains Mono (Monospace).

## ✨ Key Features

- **Animated Mesh Background**: A performant, CSS-only animated background.
- **Interactive Cursor**: A custom-coded cursor that reacts to hover states on links and buttons.
- **Glassmorphism UI**: High-end translucent cards with blur effects and thin borders.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **SDE Focused Content**: Highlighted sections for Python, C++, and Web Development expertise.

## 📂 Project Structure

- `index.html`: Contains the semantic structure and all content.
- `<style>`: Encapsulates the entire design system, animations, and media queries.
- `<script>`: Handles the custom cursor logic, scroll-to-reveal triggers, and navbar behavior.

## 🚀 How to Run

1. Clone the repository or copy the code.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, or Edge).
3. No build step or dependencies required.
